# Fix TNT Dupe Patch (Java Edition)
In Java Edition, there are a bunch of functionalities hidden within the code. This mod sets the "FIX_TNT_DUPE" flag to "true".

**Warning:** Not much information about this feature is available. Use at your own risk!

## Installation

To install this mod, place the JAR file in your "mods" directory. You must also install the Ignite mixin loader.
